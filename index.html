This lead to building a Weather Forecast WebApp. In the app the user can input the city, which they would like forecast for and also select the forecast duration. Upon sumbitting a call is made to the API for openweathermap.org, which then returns the results.

Simple CMS (IT Crowd Quotes)

I have been working with Drupal for a while now I pretty much dived into the CMS when I started development. But never really begun with a basic understanding of how Content Management Systems works in the background. So I started to build a simple CMS from the beginning using https://css-tricks.com/php-for-beginners-building-your-first-simple-cms/ as a starting place.

From there I started to improve upon this by restructuring how it worked. I created classes and then extended those classes for object-oriented programming.

I also begun going through the code and updating MySql functions set by the tutorial to MySqli.
